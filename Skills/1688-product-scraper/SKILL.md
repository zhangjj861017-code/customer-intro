# 1688 Product Scraper

从1688分销平台自动爬取符合条件的商品信息，包括商品名称、价格、近7天代发订单数和商品链接。

## 使用场景

- 从1688采集特定类别的商品数据
- 按价格范围和代发订单数筛选商品
- 获取商品的真实详情页链接
- 批量选品任务

## 输入参数

运行脚本时需要提供：

- **筛选条件** (可选)
  - `pdd`: 是否筛选拼多多支持 (true/false)
  - `baoyou`: 是否筛选包邮 (true/false)
  - `yidai`: 是否筛选一件代发 (true/false)

- **商品筛选条件**
  - `keywords`: 商品关键词列表 (如: ['网球', '羽毛球', '跳绳'])
  - `min_price`: 最低价格 (默认: 10)
  - `max_price`: 最高价格 (默认: 100)
  - `min_days7`: 近7天代发最低订单数 (默认: 1000)

- **爬取参数**
  - `target_count`: 目标商品数量 (默认: 3)
  - `max_pages`: 最多浏览页数 (默认: 20)

## 输出格式

返回符合条件的商品列表，每个商品包含：

```
商品 1:
  名称: [商品名称]
  价格: ￥[价格]
  近7天代发: [订单数]
  链接: [商品详情页URL]
```

## 工作原理

1. 打开1688分销平台
2. 应用指定的筛选条件
3. 逐页浏览商品列表
4. 按关键词和价格范围筛选
5. 提取商品的真实详情页链接
6. 返回符合条件的商品信息

## 脚本位置

主脚本: `scripts/extract_products.py`

运行方式:
```bash
python3 scripts/extract_products.py
```

## 注意事项

- 需要Playwright浏览器自动化库
- 脚本使用headless=False，会打开可见的浏览器窗口
- 如需登录，可手动扫码登录
- 爬取时间取决于页数和网络速度
