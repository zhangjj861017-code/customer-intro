# B2B企业网页构建器

为B2B企业快速生成专业的多页SPA（单页应用）介绍网页。

## 工作流程

### 1. 收集信息
- 企业名称
- 企业信息文档（.docx格式）
- 主色调（可选，默认蓝色）
- 核心产品/服务信息
- 客户案例信息

### 2. 解析文档
从.docx文件中提取：
- 公司基本信息（成立时间、团队规模、服务覆盖等）
- 产品/服务信息
- 客户案例
- 资质认证
- 竞争对手信息
- 销售渠道

### 3. 生成SPA网页
创建包含以下页面的单页应用：
- **首页**：核心优势、数据展示、动画效果
- **产品页**：产品系列详情、技术参数、应用场景
- **客户页**：核心客户案例、成功故事
- **资质页**：认证资质、专利、荣誉
- **竞争页**：竞争对手对比分析
- **渠道页**：销售渠道展示

### 4. 应用UI设计
- 专业字体系统（Poppins + Roboto）
- CSS变量色彩主题
- 流畅的页面切换动画
- 卡片悬停效果
- 响应式布局
- 每个页面独特的背景装饰

### 5. 输出HTML文件
生成完整的index.html文件，可直接部署到：
- Netlify
- Vercel
- GitHub Pages
- 任何静态网站托管服务

## 使用方式

### 基础用法
```
为我的公司创建一个介绍网页。
公司名称：派勒地垫
我有一份详细的公司信息文档（附件）
```

### 指定主色调
```
创建网页时使用绿色主题
```

### 自定义内容
```
产品页面应该重点展示这些产品...
客户案例包括...
```

## 输出格式

生成的HTML文件包含：
- 完整的HTML5结构
- 内联CSS样式（无需外部文件）
- 原生JavaScript（无框架依赖）
- 响应式设计
- 所有资源都包含在单个文件中

## 技术特点

- **无依赖**：纯HTML/CSS/JavaScript，无需构建工具
- **响应式**：支持桌面、平板、手机
- **高性能**：轻量级，快速加载
- **易部署**：单个HTML文件，直接上传即可
- **易维护**：所有内容在一个文件中，易于修改

## 配色方案

支持多种主题色：
- 蓝色系（默认）：#58a6ff
- 绿色系：#2ecc71
- 紫色系：#9b59b6
- 橙色系：#e67e22
- 红色系：#e74c3c
- 青色系：#1abc9c

## 示例

输入：企业信息.docx + 企业名称
输出：index.html（可直接在浏览器打开或部署到网络）
