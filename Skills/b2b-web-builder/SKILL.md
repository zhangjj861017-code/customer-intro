# B2B企业网页构建器 v2.0

为B2B企业快速生成专业的多页SPA（单页应用）介绍网页。

## 工作流程

### 1. 收集信息
- 企业名称
- 企业信息文档（.docx格式）
- 参考设计截图（可选）
- 主色调（可选，默认蓝色）
- 核心产品/服务信息
- 客户案例信息

### 2. 解析文档
从.docx文件中提取：
- 公司基本信息（成立时间、团队规模、服务覆盖等）
- 产品/服务信息及技术优势
- 客户案例及成功故事
- 资质认证及专利信息
- 竞争对手信息及对比分析
- 销售渠道及分布
- 组织架构及管理层
- 私域运营现状

### 3. 参照设计优化
如果提供了参考截图：
- 分析截图的UI风格、卡片设计、排版方式
- 参照配色方案和信息呈现方式
- 应用相同的交互效果和视觉层次

### 4. 生成SPA网页
创建包含以下页面的单页应用：
- **首页**：核心优势、公司信息、数据展示、动画效果
- **产品页**：产品系列详情、技术参数、应用场景、技术优势
- **客户页**：核心客户案例、成功故事、项目详情
- **资质页**：认证资质、专利、荣誉、企业认证
- **竞争页**：竞争对手对比分析、派勒优势与劣势、改进方向
- **渠道页**：销售渠道展示、线下网络、B2B优势、电商发展

### 5. 应用UI设计
- 专业字体系统（Poppins + Roboto）
- CSS变量色彩主题（蓝、绿、紫、橙、红、青）
- 流畅的页面切换动画
- 卡片悬停效果和发光边框
- 响应式布局
- 每个页面独特的背景装饰

### 6. 输出HTML文件
生成完整的index.html文件，可直接部署到：
- Netlify
- Vercel
- GitHub Pages
- 任何静态网站托管服务

## 使用方式

### 基础用法
```
为我的公司创建一个介绍网页。
公司名称：派勒地垫
我有一份详细的公司信息文档（附件）
```

### 指定主色调
```
创建网页时使用绿色主题
```

### 参照设计优化
```
我有几张参考设计截图，请参照这些截图的样式进行优化
```

### 自定义内容
```
产品页面应该重点展示这些产品...
客户案例包括...
竞争对手对比应该突出我们的优势...
```

## 输出格式

生成的HTML文件包含：
- 完整的HTML5结构
- 内联CSS样式（无需外部文件）
- 原生JavaScript（无框架依赖）
- 响应式设计
- 所有资源都包含在单个文件中

## 技术特点

- **无依赖**：纯HTML/CSS/JavaScript，无需构建工具
- **响应式**：支持桌面、平板、手机
- **高性能**：轻量级，快速加载
- **易部署**：单个HTML文件，直接上传即可
- **易维护**：所有内容在一个文件中，易于修改
- **功能完整**：导航、页面切换、动画、交互都包含

## 配色方案

支持多种主题色：
- 蓝色系（默认）：#58a6ff - 专业科技感
- 绿色系：#2ecc71 - 环保生态感
- 紫色系：#9b59b6 - 创意创新感
- 橙色系：#e67e22 - 活力热情感
- 红色系：#e74c3c - 商业竞争感
- 青色系：#1abc9c - 清爽现代感

## 页面结构

每个页面都包含：
- 独立的背景渐变
- 主题色的标题和按钮
- 卡片式内容展示
- 详情内容区域
- 返回首页按钮

## 内容组织

### 首页
- Hero区域：公司名称、slogan、CTA按钮
- 核心优势：6个关键优势卡片
- 公司信息：成立时间、团队规模、服务覆盖
- 联系方式：电话、邮箱、地址

### 产品页
- 产品系列：6个产品卡片
- 技术优势：专利、认证、资质
- 应用案例：典型项目详情

### 客户页
- 核心客户：3-6个客户案例
- 项目详情：成功故事、成果数据

### 资质页
- 认证资质：ISO、高新企业等
- 专利信息：20+专利详情
- 荣誉认可：行业奖项

### 竞争页
- 竞争对手对比表
- 派勒优势分析
- 劣势与改进方向

### 渠道页
- 销售渠道卡片
- 线下销售网络
- B2B渠道优势
- 电商发展方向

## 最佳实践

1. **内容准备**：提供完整的企业信息文档
2. **参考设计**：如有参考截图，一起提供以获得更好的UI效果
3. **内容审核**：生成后检查所有内容是否准确
4. **部署测试**：部署后在不同设备上测试响应式效果
5. **后续维护**：所有内容都在HTML中，可直接编辑修改
